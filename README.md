# SQL Practice

データ抽出・集計のためのSQL学習記録とクエリ集です。
基本的な構文から、実務で使える複雑なデータ操作までを継続的に書き溜めています。

## 学習環境 (Environment)
* 実行環境:Google Colaboratory 

## ディレクトリ構成 (Contents)
* `01_basics/` : SELECT, WHERE, JOIN などの基本構文
* `02_aggregation/` : GROUP BY, HAVING を用いた集計処理
* `03_advanced/` : サブクエリ、CTE（WITH句）、ウィンドウ関数（ROW_NUMBER, RANK等）
* `exercises/` : 練習問題や競技SQLの解答コード
